快速啟動本機服務

擴充無法代為執行本機程式。請複製指令到終端機；模型路徑請改為你的檔案。詳見擴充套件內 docs/LOCAL_SETUP.zh-TW.md。

WHISPER SERVER (8080) 
whisper-server -m "C:/path/to/ggml-base.bin" --host 127.0.0.1 --port 8080
OLLAMA CORS ALLOW (POWERSHELL) 
$env:OLLAMA_ORIGINS="chrome-extension://*"; ollama serve
